# Images pour le blog iT-REXpert

## Logo rex-logo.png

Pour créer votre logo de dinosaure T-Rex, vous pouvez :

### Option 1 : Générer avec un outil en ligne
- Canva (https://www.canva.com) - Chercher "T-Rex logo" ou "dinosaur logo"
- Logo.com
- Looka.com

### Option 2 : Utiliser une image libre de droits
- Unsplash (https://unsplash.com/s/photos/t-rex)
- Pexels (https://www.pexels.com/search/dinosaur/)
- Pixabay (https://pixabay.com/images/search/t-rex/)

### Option 3 : Créer avec DALL-E, Midjourney ou Stable Diffusion
Prompt suggéré :
```
"T-Rex dinosaur logo in Jurassic Park style, green and orange colors,
tech theme, minimalist, professional, IT expert branding"
```

### Spécifications recommandées
- Format : PNG avec transparence
- Dimensions : 300x300px minimum
- Couleurs : Vert jungle, orange/rouge Jurassic Park
- Style : Minimaliste mais impactant

## Favicon
Le même logo peut être utilisé pour le favicon (redimensionné en 32x32px).
